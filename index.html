FileFinder

A simple command-line tool to help you find files and directories on your system when you forget their location.

FileFinder Tool Features

  • Search for files or directories by name.
  • Fast and efficient search.
  • Built with Rust for performance and reliability.

How to Install

  1. Clone the repository: git clone https://github.com/dorsabatenie/filefinder.git
  2. Navigate to the project directory: cd filefinder
  3. Build the project: cargo build --release

Ready to find your files? Start using FileFinder today! It's quick, easy, and built for reliability.
