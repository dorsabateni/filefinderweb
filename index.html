Installation Instructions

Getting started with FileFinder is easy. Follow the steps below to have it up and running:

1. Clone the Repository

git clone https://github.com/dorsabatenie/filefinder.git

2. Navigate to the Project Directory

cd filefinder

3. Build the Project

cargo build --release
